# Sistema de Reservas RFLVG

Este es un sistema de gestión de turnos y reservas desarrollado con **Python** y **Flask**. El proyecto presenta una interfaz minimalista en modo oscuro (Dark Mode), orientada a empresas que buscan una estética limpia y profesional.

## Características

* **Interfaz de Usuario:** Formulario de reserva responsivo con validaciones.
* **Panel Administrativo:** Visualización ordenada de reservas con opción de eliminación.
* **Base de Datos:** Persistencia de datos local utilizando **SQLite3**.
* **Estética:** Diseño minimalista basado en una paleta de colores oscura y acentos en tono tierra.

## Tecnologías utilizadas

* **Backend:** Python 3.x, Flask.
* **Frontend:** HTML5, CSS3 (Custom styles).
* **Base de Datos:** SQLite.
